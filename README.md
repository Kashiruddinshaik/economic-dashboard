#  Economic Dashboard – Streamlit App

This project is a dashboard to visualize economic indicators using World Bank data.  
You can select a country, view key indicators over time, see smart insights, and get GDP forecasting using Prophet.

---

##  Features

- Select country and indicator from sidebar
- Line chart of indicator over time
- Shows latest value + % change since 2000
- Forecast GDP (2024–2027) using Facebook Prophet
- Download raw data and forecast as CSV
- Clean UI with metric box and summary insight
